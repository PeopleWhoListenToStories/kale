### ✨ 特性

- 🛠️ &nbsp; 提供日常开发中经常使用的实用工具
- 🛠️ &nbsp; 工具实现非常轻量
- 🛠️ &nbsp; 使用 ts 编写，提供完善的类型支持
- 💪 &nbsp; 确保 99% 以上单元测试覆盖率，提供稳定性保证

---

### 📦 安装

```shell
# npm
npm i @kale/kit -S
# yarn
yarn add @kale/kit
# pnpm
pnpm add @kale/kit
```

### 🚀 使用

```ts
import { isString } from '@kale/kit's

isString('@kale/kit') // return true
isString(123) // return false
```
