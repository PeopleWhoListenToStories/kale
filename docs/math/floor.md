# floor

Return number rounded down to precision.

### Usage

```ts
import { floor } from '@kale/kit'

floor(1.005) // return 1
floor(1.0015, 3) // return 1.001
```

### Arguments

| Arg         |   Type   | Defaults |
| ----------- | :------: | -------: |
| `val`       | `number` |          |
| `precision` | `number` |      `0` |

### Return

|   Type   |
| :------: |
| `number` |
