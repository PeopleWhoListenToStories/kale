# ceil

Return number rounded up to precision.

### Usage

```ts
import { ceil } from '@kale/kit'

ceil(1.004) // return 2
ceil(1.004, 2) // return 1.01
```

### Arguments

| Arg         |   Type   | Defaults |
| ----------- | :------: | -------: |
| `val`       | `number` |          |
| `precision` | `number` |      `0` |

### Return

|   Type   |
| :------: |
| `number` |
