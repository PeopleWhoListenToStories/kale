# at

获取 `数组` 中指定索引的元素，支持负索引。

### 使用

```ts
import { at } from '@kale/kit'

at([1, 2, 3], 0) // return 1
at([1, 2, 3], -1) // return 3
```

### 参数

| 参数    | 类型     | 默认值 |
| ------- | -------- | ------ |
| `arr`   | `Array`  |        |
| `index` | `number` |        |

### 返回值

| 类型  |
| ----- |
| `any` |
