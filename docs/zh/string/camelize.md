# camelize

将字符串转换为 `camelCase` 格式。

### 使用

```ts
import { camelize } from '@kale/kit'

camelize('hello-world') // return 'helloWorld'
camelize('FooBar') // return 'fooBar'
```

### 参数

| 参数    |   类型   | 默认值 |
| ------- | :------: | -----: |
| `value` | `string` |        |

### 返回值

|   类型   |
| :------: |
| `string` |
