# randomString

生成指定长度的随机字符串。

### 使用

```ts
import { randomString } from '@kale/kit'

randomString() // 生成一个长度为 10 的随机字符串
randomString(30) // 生成一个长度为 30 的随机字符串
```

### 参数

| 参数     |   类型   | 默认值 |
| -------- | :------: | -----: |
| `length` | `number` |   `10` |

### 返回值

|   类型   |
| :------: |
| `string` |
