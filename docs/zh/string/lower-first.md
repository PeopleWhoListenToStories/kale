# lowerFirst

将 `字符串` 的 `首字母` 小写，其余部分保持不变。

### 使用

```ts
import { lowerFirst } from '@kale/kit'

upperFirst('Hello') // return 'hello'
upperFirst('@kale/kit') // return '@kale/kit'
```

### 参数

| 参数    |   类型   | 默认值 |
| ------- | :------: | -----: |
| `value` | `string` |        |

### 返回值

|   类型   |
| :------: |
| `string` |
