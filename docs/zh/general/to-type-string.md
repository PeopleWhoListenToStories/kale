# toTypeString

返回输入值的 `类型字符串`。

### 使用

```ts
import { toTypeString } from '@kale/kit'

toTypeString({}) // return '[object Object]'
toTypeString([]) // return '[object Array]'
```

### 参数

| 参数    | 类型  | 默认值 |
| ------- | :---: | -----: |
| `value` | `any` |        |

### 返回值

|   类型   |
| :------: |
| `string` |
