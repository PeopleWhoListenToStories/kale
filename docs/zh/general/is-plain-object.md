# isPlainObject

判断输入值是否是 `普通对象`。

### 使用

```ts
import { isPlainObject } from '@kale/kit'

isPlainObject({}) // return true
isPlainObject([]) // return false
```

### 参数

| 参数    | 类型  | 默认值 |
| ------- | :---: | -----: |
| `value` | `any` |        |

### 返回值

|   类型    |
| :-------: |
| `boolean` |
