# sumBy

根据提供的函数计算 `数组` 中各项值的和。

### 使用

```ts
import { sumBy } from '@kale/kit'

sumBy([{ n: 1 }, { n: 2 }, { n: 3 }], ({ n }) => n) // return 6
```

### 参数

| 参数  | 类型                   | 默认值 |
| ----- | ---------------------- | ------ |
| `arr` | `Array`                |        |
| `fn`  | `(val: any) => number` |        |

### 返回值

| 类型     |
| -------- |
| `number` |
