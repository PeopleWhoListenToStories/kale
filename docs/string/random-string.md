# randomString

Generate a random alphanumeric string of a specified length.

### Usage

```ts
import { randomString } from '@kale/kit'

randomString() // Generates a random string of length 10
randomString(30) // Generates a random string of length 30
```

### Arguments

| Arg      |   Type   | Defaults |
| -------- | :------: | -------: |
| `length` | `number` |     `10` |

### Return

|   Type   |
| :------: |
| `string` |
