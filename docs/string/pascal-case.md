pascalCase

Convert a string to `PascalCase`.

### Usage

```ts
import { pascalCase } from '@kale/kit'

pascalCase('hello-world') // return 'HelloWorld'
pascalCase('fooBar') // return 'FooBar'
```

### Arguments

| Arg     |   Type   | Defaults |
| ------- | :------: | -------: |
| `value` | `string` |          |

### Return

|   Type   |
| :------: |
| `string` |
