# lowerFirst

Lowercase the `first letter` of the `string` and keep the rest unchanged.

### Usage

```ts
import { upperFirst } from '@kale/kit'

upperFirst('Hello') // return 'hello'
upperFirst('@kale/kit') // return '@kale/kit'
```

### Arguments

| Arg     |   Type   | Defaults |
| ------- | :------: | -------: |
| `value` | `string` |          |

### Return

|   Type   |
| :------: |
| `string` |
