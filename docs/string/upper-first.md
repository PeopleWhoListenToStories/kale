# upperFirst

Capitalize the `first letter` of a `string`, leaving the rest of the string unchanged.

### Usage

```ts
import { upperFirst } from '@kale/kit'

upperFirst('hello world') // return 'Hello world'
upperFirst('@kale/kit') // return '@kale/kit'
```

### Arguments

| Arg     |   Type   | Defaults |
| ------- | :------: | -------: |
| `value` | `string` |          |

### Return

|   Type   |
| :------: |
| `string` |
