# preventDefault

Prevents the default action of an event if it is cancelable.

### Usage

```ts
import { preventDefault } from '@kale/kit'

document.addEventListener('click', preventDefault)
```

### Arguments

| Arg     | Type    | Defaults |
| ------- | ------- | -------- |
| `event` | `Event` |          |

### Return

| Type   |
| ------ |
| `void` |
