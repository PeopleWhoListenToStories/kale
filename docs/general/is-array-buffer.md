# isArrayBuffer

Determine whether the input value is a `ArrayBuffer`.

### Usage

```ts
import { isArrayBuffer } from '@kale/kit'

isArrayBuffer(new ArrayBuffer(8)) // return true
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
