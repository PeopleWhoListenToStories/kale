# isRegExp

Determine whether the input value is a `RegExp`.

### Usage

```ts
import { isRegExp } from '@kale/kit'

isRegExp(/abc/) // return true
isRegExp('abc') // return false
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
