# isBoolean

Determine whether the input value is a `boolean`.

### Usage

```ts
import { isBoolean } from '@kale/kit'

isBoolean(true) // return true
isBoolean('@kale/kit') // return false
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
