# isWeakMap

Determine whether the input value is a `WeakMap`.

### Usage

```ts
import { isWeakMap } from '@kale/kit'

isWeakMap(new WeakMap()) // return true
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
