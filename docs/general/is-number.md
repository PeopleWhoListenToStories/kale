# isNumber

Determine whether the input value is a `number`.

### Usage

```ts
import { isNumber } from '@kale/kit'

isNumber(123) // return true
isNumber('@kale/kit') // return false
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
