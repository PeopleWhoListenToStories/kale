# isEqual

Deeply compare two values.

### Usage

```ts
import { isEqual } from '@kale/kit'

isEqual({ n: 1 }, { n: 1 })
// return true
isEqual([1, 2, 3], [1, 2, 3])
// return true
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |
| `other` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
