# isTypedArray

Determine whether the input value is a `TypedArray`.

### Usage

```ts
import { isTypedArray } from '@kale/kit'

isTypedArray(new Int8Array(8)) // return true
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
