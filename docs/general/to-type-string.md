# toTypeString

Return the `type string` of the input value.

### Usage

```ts
import { toTypeString } from '@kale/kit'

toTypeString({}) // return '[object Object]'
toTypeString([]) // return '[object Array]'
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type   |
| :------: |
| `string` |
