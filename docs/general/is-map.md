# isMap

Determine whether the input value is a `Map`.

### Usage

```ts
import { isMap } from '@kale/kit'

isMap(new Map()) // return true
isMap({}) // return false
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
