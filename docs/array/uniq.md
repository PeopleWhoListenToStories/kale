# uniq

Creates a duplicate-free version of an `array`, using the values equality.

### Usage

```ts
import { uniq } from '@kale/kit'

uniq([1, 2, 2, 3]) // return [1, 2, 3]
uniq(['a', 'a', 'b', 'c']) // return ['a', 'b', 'c']
```

### Arguments

| Arg   | Type    | Defaults |
| ----- | ------- | -------- |
| `arr` | `Array` |          |

### Return

| Type    |
| ------- |
| `Array` |
