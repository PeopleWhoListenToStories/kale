# xor

XOR(Exclusive OR) the passed array and return a new `array`.

### Usage

```ts
import { xor } from '@kale/kit'

xor([1, 2, 2], [1, 3])
// return [2, 3]
xor([1, 2], [1, 2], [3])
// return [3]
```

### Arguments

| Arg         | Type           | Defaults |
| ----------- | -------------- | -------- |
| `...values` | `Array<Array>` |          |

### Return

| Type    |
| ------- |
| `Array` |
